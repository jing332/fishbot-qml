# Fishbot-QML

![Build qmake](https://github.com/jing332/fishbot-qml/workflows/Build%20qmake/badge.svg)

Fishbot-QML是一款由QML+Go开发的GUI程序,
基于Minecraft Golang协议库[Tnze/go-mc](https://github.com/Tnze/go-mc)

# 功能

- [x] 服务器管理
- [x] 账户管理
- [x] 加入Minecraft服务器
